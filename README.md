##Environment Setup

####1. Install Node
	npm --version
	
####2. Install Gulp
	npm install -g gulp
	
####3. Install Ruby
	gem update
	
####4. Install Browserify
	npm install -g browserify
	
####5. Install Compass
	gem install compass
	
####6. Install Sass
	gem install sass
	
###Create Package.JSON
	npm init
